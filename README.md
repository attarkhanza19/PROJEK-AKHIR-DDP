# DESKRIPSI SINGKAT
Program ini dibuat untuk mempermudah pengguna dalam memesan dan memanajemen sebuah konser, dan juga mempermudah pelayanan, penyediaan, dan pendataan transaksi kepada pelaku bisnis(admin). Program ini memiliki beberapa fitur, yaitu dapat melakukan registrasi, login, pengecekan saldo, pengisian saldo, pembelian tiket, menambahkan konser, menampilkan harga dan daftar acara konser  yang tersedia, melihat apa aja daftar tiket yang sudah dibeli, melihat history konser, kembali dan keluar untuk CRUD (create, update, read, delete), kembali dan keluar untuk admin.

Pada fitur ini memiliki ketentuan yang harus di ikuti. Ketentuannya adalah
1. Format tanggal harus DD-MM-YY
2. Jika memasukan tanggal yang tidak sesuai (misal 40-13-2025 atau memasukan selain angka) akan keluar "Format tanggal tidak valid!"
3. Format waktu HH:MM
4. Jika memasukkan waktu yang tidak sesuai (misal 25:00 atau memasukkan selain angka) akan keluar "Format waktu tidak valid!"
5. Harga minimal tiket VVIP 100.000 dan maksimal 10.000.000
6. Harga minimal tiket VIP 70.000 dan maksimal 9.000.000
7. Harga tiket VIP tidak boleh lebih tinggi dari VVIP
8. Harga minimal tiket festival 50.000 dan maksimal 8.000.000
9. Harga tiket Festival tidak boleh lebih tinggi dari VIP dan VVIP
10. Input tidak boleh negatif
11. Tidak bisa menekan CTRL + C
